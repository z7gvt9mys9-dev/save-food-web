## Обзор
Save Food Web — это мост между теми, у кого есть лишняя еда (кафе, магазины), и теми, кому она нужна. Наша миссия — снизить объем пищевых отходов с помощью технологий.

### Ключевой функционал:
*  **Интерактивная карта:** Поиск ближайших точек раздачи в реальном времени.
*  **Ролевая модель:** Доноры (бизнес), Получатели, Доставщики и Администраторы.
*  **Smart Notifications:** Мгновенные уведомления о новых лотах рядом с вами.
*  **Dashboard:** Личная статистика спасенной еды и достижений (XP).
